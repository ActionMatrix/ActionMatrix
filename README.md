# 一、简介

ElectronTodo是一个基于Electron的待办事项几日程管理工具。

# 二、快速开始

1. 安装Node.js依赖包

    ```shell
    yarn
    ```

2. 编译、开发

    ```shell
    yarn electron:serve
    ```

3. 构建安装包

    ```shell
    yarn electron:build
    ```
